Email Sending (GitHub Actions)

Set these as GitHub Actions secrets — the workflow handles sending

⚡ These settings are stored as GitHub Actions secrets, not here. Follow the steps below to configure email sending.
1
Enable 2-Step Verification on your Google account at myaccount.google.com/security
2
Generate a Gmail App Password at myaccount.google.com/apppasswords. Select app: Mail, device: Other.
3
Go to your repo → Settings → Secrets and variables → Actions and add two secrets:
  • GMAIL_USER — your Gmail address
  • GMAIL_PASS — the 16-character App Password
4
Done! The workflow at .github/workflows/birthday-check.yml runs every day at 8:00 AM UTC and sends birthday emails automatically. 🎉
